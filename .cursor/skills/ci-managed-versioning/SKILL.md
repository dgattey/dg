# CI-managed versioning

- Never change version fields manually.
- Version bumps and release versions are generated by CI.
- Before committing or pushing, inspect the diff for version-only changes and restore them to the target branch value.
- Do not create version-bump commits, tags, or releases unless Dylan explicitly asks to bypass CI for that specific operation.
